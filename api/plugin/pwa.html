# @mr-hope/vuepress-plugin-pwa 新增

Progressive Web App Surpport

# 插件说明

开启 PWA 支持。当获得更新时将会在右下角显示弹窗。

弹窗默认语言将自动设置为 themeConfig 中配置的 baseLang。

该功能默认启用，如果你不需要该功能或想使用其他的 pwa 插件，可以设置 themeConfig.pwa 为 false 来禁用本插件。
