# Tex 语法支持

让你的 VuePress 站点中的 Markdown 文件支持 TeX\TeXTE​X 语法。

# 配置



 

 
 



module.exports = {
  themeConfig: {
    markdown: {
      // 启用 TEX 支持
      tex: true
    }
  }
};
